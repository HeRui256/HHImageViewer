Chinese&中文
本程序是由个人开发者开发的图片查看器 《HH查图器》 （英文名HHImageViewer）

>>>使用说明
将图片和此文件放在同一个文件夹，运行本文件以查看同目录下的文件图片
如果同时存在多个图片，按↑或←以切换上一张图片，按↓或→切换下一张图片（文件顺序按文件名排序文件）
调整图片大小时，需要调整窗口的横向宽度，调整宽度后，再点击图片一下就可以调整图片，（也可以按快捷键Shift或Ctrl或Alt或Backapace或Enter或空格）
按Esc键退出程序

>>>最佳使用方法：将此程序放在桌面，以查看桌面上的图片

---------*- 后面是废话 -*---------
本程序界面极度精简（除了图片啥都木有），可用于画师绘图用
使用Python编写，调用tkinter库实现图形化界面，调用PIL库以加载图像
调用库：base64，os，PIL，tkinter，zlib
单文件脚本，不需要额外的文件

>>>稳定支持的格式（经过测试，能正常打开下列格式）
png，jpg，gif，bmp，jpeg，tga
>>>理论上能支持的格式（没测试过，本人没见过这些格式，但下列格式PIL库都支持，那理论上就都可以）
blp，bufr，cur，dcx，dds，grib，ico，im，jfif，jpe，jpx，jpf，apng，psd，rgb，tif，tiff，webp，xpm，xbm

English&英文 (machine translation&机翻)
>>> instructions for use
Place the picture in the same folder as this file and run the file to see the picture of the file in the same directory
If there are multiple images at the same time, press ↑ or ← to toggle the previous image, press ↓ or → to switch the next image (file order sort files by file name)
When you adjust the size of the picture, you need to adjust the horizontal width of the window, after adjusting the width, you can adjust the picture with one click on the picture, (you can also press the shortcut key Shift or Ctrl or Alt or Backapace or Enter or space)
Press escape to exit the program

>>> best practice: Place this program on your desktop to see a picture on your desktop

---------* - Followed by nonsense -*---------
The interface of this program is extremely streamlined (except for pictures), which can be used by painters for drawing
Written in Python, the tkinter library is called to implement a graphical interface, and the PIL library is called to load images
Calling libraries: base64, os, PIL, tkinter, zlib
Single-file script, no additional files required

>>> stable supported formats (tested to open the following formats normally)
png，jpg，gif，bmp，jpeg，tga
>>> formats that can theoretically be supported (I have not tested, I have not seen these formats, but the following formats are supported by the PIL library, then theoretically it can be)
blp，bufr，cur，dcx，dds，grib，ico，im，jfif，jpe，jpx，jpf，apng，psd，rgb，tif，tiff，webp，xpm，xbm
